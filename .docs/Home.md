## System Overview
The `socket-data-analyzer-c` is a high-performance C-based pipeline designed to process streaming video/audio data chunks from ESP32-S3 devices. It handles gap detection, log parsing, and efficient storage of clip metadata.

Refer to the [[Full System Spec|full_spec]] for architectural details.
